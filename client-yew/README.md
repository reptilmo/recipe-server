# client-yew
Yew client for the recipe server.


### Building
Install WebAssembly target.
```
    rustup target add wasm32-unknown-unknown
```
Install `trunk`.
```
    cargo install trunk
```
Run trunk to serve the client.
```
    trunk serve
```
